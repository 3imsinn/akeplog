Easily add an author box bio signature with custom profile image and social profile buttons to the end of each author's posts

== Description ==

DT Author Box is great for those who have multiple authors on a Wordpress site and want to show author information at the bottom of every post for each author like a signature. This can include an image, a short bio, a link to the author's website and their social profiles.
